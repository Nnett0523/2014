Jurad

ALE1.RF.012
Archivo de Jurad. Se debe cargar el archivo en una estructura de datos de tal forma que quede disponible para las alertas del sistema en línea y para Triggers (alertas implementadas en Teradata).

Nota: El motor de alertas se especializó para generar todas las alertas.

ALE1.RF.013
El archivo suministrado se actualizará periódicamente, con una frecuencia que puede ser diaria, semanal o mensual.

La actualización de la información debe poder hacerse en caliente de tal forma que no se requiera bajar los servicios para que la nueva información sea tenida en cuenta

ALE.RNF.002
Desarrollar un programa de Cargue universal para los archivos como Jurad y otras fuentes externas
