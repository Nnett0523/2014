Ticket burnout

Calculating the numbers

The calculation is based on an object X containing the values of a row, as X.data.['col header'].

On initialization this object is furnished with a list of (counter, condition) items.

Each of these conditions is calculated with the current row values and, if true, the associated counter is incremented.

The process loops over the input data rows and for each one it sets the current row values into the X object and loops over the (counter, condition) items incrementong the counters each time a condition happens to be true.

In addition to the input row data, there are a few calculated items, like if the ticket date belongs to this week, and if so the weekday number (Mon, Tue, ...):

  • isInCurrentWeek
  • isInCurentMonth
  • burntByMonday
  • burntByTuesday
  • burntByWednesday
  • burntByThursday
  • burntByFriday

The (counter, condition) list

B Target
Target semanal de tickets pendientes, 20 por día excepto feriados. En el ejemplo, el lunes es feriado. 

D Acc. Staffed
Acumulado desde el lunes al día que corresponde a la fila, de los tickets cerrados al final de cada día, contando los tickets con:
H-Client ≠ “Globant” y J-Type of Position = “CONFIRMED” y L-Replacement = “No” (en el cuadro: D5=F5+G5)

C Actual
Diferencia entre C4 y D4. Recordar que D4 debe sumar el día actual y los días anteriores.

F Hired
C-Status = “Hired” y H-Client ≠ “Globant” y L-Replacement = “No” y J-Type of Position = “CONFIRMED” [NG] Para cada día

G Staffed
C-Status = “Internally Staffed” y H-Client ≠ “Globant” y L-Replacement = “No” y J-Type of Position = “CONFIRMED” [NG] Para cada dia

E Onsite staffed
K-Work Office starts with “EU” or K-Work Office starts with “UK”
4E burntByMonday && ( this.data['Work Office'].substr(0, 2) === 'EU' || this.data['Work Office'].substr(0, 2) === 'UK' ) 
5E burntByTuesday && ( this.data['Work Office'].substr(0, 2) === 'EU' || this.data['Work Office'].substr(0, 2) === 'UK' ) 
6E burntByWednesday && ( this.data['Work Office'].substr(0, 2) === 'EU' || this.data['Work Office'].substr(0, 2) === 'UK' ) 
7E burntByThursday && ( this.data['Work Office'].substr(0, 2) === 'EU' || this.data['Work Office'].substr(0, 2) === 'UK' ) 
8E burntByFriday && ( this.data['Work Office'].substr(0, 2) === 'EU' || this.data['Work Office'].substr(0, 2) === 'UK' )
