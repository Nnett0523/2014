moodle plugins and blocks

CLOZE Editor
An interface that integrates with Moodles existing HTML editor (HTMLArea in 1.9 and TinyMCE in 2.x) to support teachers in creating CLOZE (Embedded answers) questions in the GIFT syntax. It will also read and edit existing CLOZE items.
HTML5 content editable editor
Replacement text editor that uses HTML5 contenteditable regions to create a simpler and faster text editor. If you find any issues, please report them on github.
Blocks: Course Publishing Workflow
Publishflow reveals a string MNET based course publishing architecture for providing a high level service for moving, cataloging, preproducing courses and organize several moodles as a courseware production supersystem.
Blocks: Attendance
This block works with the attendance module, it provides teachers with quick access to Attendance functionality, and provides students quick access to a summary report for their own attendance.
Blocks: Course contents
Course contents block produces a table of contents for the course - that is a list of all visible sections (topics or weeks) in the course. Clicking at one of these links will display that particular section.
