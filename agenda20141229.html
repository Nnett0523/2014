Security configuration

Minimum of single factor auhentication: user & password.

If there is personal information or monet movement involved then authentication must be multi-factor (like with RSA token).

Password configuration:

  • minimum 8 characters
  • expiry in 90 days
  • password complexity
  • accuont disabled upon inactivity
  • account locking after 5 consecutive logon attempts

Logging enabled for logins, actions, timestamps, regulatory transactions, critical foelds.

Log out terminates the session completely, removes cached elements from the server.

OS outside access has to be authorized.

The OS is kept updated with the latest patches and updates.
