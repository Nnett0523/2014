Experian through Firefox

Ubuntu: Use SSL Network Extender

Get a 32-bit Firefox version

As of now FF29 is the latest and newest, in the future other will come ...

https://ftp.mozilla.org/pub/mozilla.org/firefox/releases/29.0b9/win32/en-US/

Allow renegotiation for the Experian portal

See http://wiki.mozilla.org/Security:Renegotiation:

in about:config set

security.ssl.renego_unrestricted_hosts = portal.computec.com.co

It gives an error "Please provide a valid client-certificate"

Import the certificate into Firefox

  • Open: menu → options → advanced → certificates → view certificates → your certificates → [import]
  • Choose the certificate file (the one with extension .p12)
  • Provide the certificate encryption password
  • Do the normal steps (as explained in Experian's instructions document)
  • In: menu → options → advanced → certificates choose "select one automatically"

Connection

  • Open portal.computec.com.co
  • Choose "Certificate Sign In" and click [Sign In]
  • Click [Connect]
  • If the browser wants to update Java, do it using this link, else it will bug you forever
  • Accept to run the "Check Point Deployment Agent", you might want to check the "Do not whow again ..." option
  • Enter the password you assigned to the certificate when you imported it (?)
