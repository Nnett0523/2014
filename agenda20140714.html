Installing Anchor

Options -indexes

<IfModule mod_rewrite.c>
RewriteEngine On
RewriteBase /a

# Allow any files or directories that exist to be displayed directly
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d

# Rewrite all other URLs to index.php/URL
RewriteRule ^(.*)$ index.php/$1 [L]
</IfModule>

<IfModule !mod_rewrite.c>
ErrorDocument 404 index.php
</IfModule>

Nautilus as root

Added a launcher "Nautilus as root" thus: 

#!/usr/bin/env xdg-open

[Desktop Entry]
Version=1.0
Type=Application
Terminal=false
Icon[en_US]=/home/jlanus/bin/icos/nautilusRoot.png
Exec=gksudo nautilus /
Comment[en_US]=Use with care
Name[en_US]=Nautilus as root
Name=Nautilus as root
Comment=Use with care
Icon=/home/jlanus/folder

The command part came from a StackOverflow arficle "How do I start Nautilus as root?".

I used my launcher builder.

For the icon, I downloaded a nautilus.png icon and slightly modified it, see the image.
