WAMP es el acrónimo usado para describir un sistema de infraestructura de internet que usa las siguientes herramientas:

  • Windows, como sistema operativo;

  • Apache, como servidor web;

  • MySQL, como gestor de bases de datos;

  • PHP (generalmente), Perl, o Python, como lenguajes de programación.

El uso de un WAMP permite servir páginas html a internet, además de poder gestionar datos en ellas, al mismo tiempo un WAMP, proporciona lenguajes de programación para desarrollar aplicaciones web.

Tomado de: http://es.wikipedia.org/wiki/WAMP

Ayuda para instalar: http://www.wedeveloped.com/index.php/articulos-y-tutoriales/software/item/34-wamp-instalar-servidor-apache-mysql-y-php-en-windows-7

PHPMYADMIN es una herramienta escrita en PHP con la intención de manejar la administración de MySQL a través de páginas web, utilizando Internet. Actualmente puede crear y eliminar Bases de Datos, crear, eliminar y alterar tablas, borrar, editar y añadir campos, ejecutar cualquier sentencia SQL, administrar claves en campos, administrar privilegios, exportar datos en varios formatos.Tomado de: http://es.wikipedia.org/wiki/PhpMyAdmin
