80100-VALIDAR-IDENTIFICACION

  1. Inicializa y valida los parámetros de la llamada
  2. Según diversos switches, deduce el tipo de ID a partir del número, valida el tipo de ID según el número, ...
  3. Valida el número de identificación contra los archivos de REGISTRADURIA, DIAN y DAS. Si es una persona calcula un nombre armado a partir de los info de la base de datos.
  4. Si SW-VERIFNOM-BDIIVIDE = 1 hace el matching de nombres usando 10000-MATCH-NOMBRES-VALNOM y si no es OK retorna SW-NOMBRES-BDIIVIDE
