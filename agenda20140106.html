Pages for year 2014

It contains HTML pages, one for each day of the year, and a "res" directory with resources.

I use these pages to store daily notes and links, editing them with W3C's Amaya editor and browser. Amaya is good because it is so WYSIWYG, that is, there is no difference between the browse and the edit views, it's not plagued with editor tools. The downside is that Amaya development was stopped years ago ...

The res directory contains a script that generates the pages after a model, it's a js program that runs in node.js. There is also an Excel file that might be used too.

This created two files in my /c/Users/--me--/.ssh folder. These files are my private and public keys (id_rsa and id_rsa.pub). The public key is the one we will be moving to remote git repository servers.

Explains how to paste the public key into the server:

Git For Windows Developers – Git Series – Part 2

 

The steps are:

  1. create the keys pair with ssh-keygen -C "juan.lanus@gmail.com" -t rsa in directory ~/.ssh
  2. copy the content of the ...pub file in the clipboard
  3. in github edit profile and choose "SSH Keys"
  4. paste the previously copies key and accept

After this the git push origin master command works fine.
