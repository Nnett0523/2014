installing PHP

  • Setting Apache to start as a service: sudo update-rc.d apache2 enable
  • Apache error log:/var/log/apache2/error.log
  • Apache configuration: /etc/apache2/apache2.conf
  • Apache restart: sudo apache2ctl restart (also stop and start)
  • Apache didn't know the server name and threw an error each sime it started: added to the configuration file 000-default.conf: 
    ServerName localhost
