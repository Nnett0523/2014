Static & Dynamic SQL in COBOL :: A Walk-Through 

* assign query to variable.
MOVE 'SELECT NAME, ADDRESS FROM STUDENT' TO WS-SQL-COMMAND.
* prepare SELECT command.
     EXEC SQL 
         PREPARE Q1 FROM :WS-SQL-COMMAND 
     END-EXEC
* declare cursor for SELECT command. 
     EXEC SQL 
         DECLARE C1 CURSOR FOR Q1
     END-EXEC
* open cursor. 
     EXEC SQL 
         OPEN C1 
     END-EXEC
* fetch data. 
     EXEC SQL 
         FETCH C1 INTO :WS-NAME, :WS-ADDRESS 
     END-EXEC 
* close cursor. 
     EXEC SQL 
         CLOSE C1 
     END-EXEC
