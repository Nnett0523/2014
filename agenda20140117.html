Contributing to Ubuntu (SpecSpec)

Created an icon builder icon in the desktop

There is an Ubuntu article "Using gnome-panel/alacarte".

Unity Launchers are actually files stored in your computer (in ~/Desktop, for example), with a '.desktop' extension.

The icon creator is made thus: 

 gnome-desktop-item-edit ~/Desktop/ --create-new

The resulting file might not work, check it with

desktop-file-validate makeLauncher

The validator is documented in this page.

The problem (as of 13.10) was that the GUI produced a line

GenericName[en_US]=make launcher

that was not accepted by the system. The validator complained about it.
